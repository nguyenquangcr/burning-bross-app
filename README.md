# Get started

```sh
  git clone https://github.com/nguyenquangcr/burning-bross-app.git

  # Clone .env file
  yarn install

  # Serve dev mode
  yarn dev

  # Generate a commit
  yarn commit

  # Build production mode
  yarn build

```
